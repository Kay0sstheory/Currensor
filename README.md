# Curren$or 💱

A sleek little currency converter you can self-host in seconds. Compare up to 5 currencies at once, swap them around, search through 20+ options. One HTML file, zero setup.

## Features

- **Multi-currency view** — See up to 5 conversions at once, side by side
- **Smart swap logic** — Picking a currency that's already in use auto-swaps instead of duplicating
- **Bottom-sheet picker** — Mobile-friendly currency selector with instant search
- **20 currencies** — All major world currencies plus key emerging markets
- **Real-time input** — Results update as you type, non-numeric characters filtered out
- **Zero dependencies** — One HTML file. No frameworks, no build step, no node_modules

## How to Use

**Option 1: Just open it**
```bash
git clone https://github.com/Kay0sstheory/Currensor.git
cd Currensor
# Open index.html in your browser — that's it
```

**Option 2: GitHub Pages** *(free hosting)*
1. Go to your repo → Settings → Pages
2. Under "Source", select `main` branch
3. Your converter will be live at `https://kay0sstheory.github.io/Currensor/`

## Supported Currencies

USD, EUR, GBP, JPY, CAD, AUD, CHF, CNY, INR, MXN, BRL, KRW, SGD, SEK, NOK, NZD, ZAR, TRY, AED, THB

## Tech Stack

- **Pure HTML, CSS, JavaScript** — no frameworks, no libraries
- **DM Sans + Space Mono** — fonts loaded from Google Fonts
- **Dark theme** with noise texture and accent glow effects
- Mobile-responsive down to 400px viewport
